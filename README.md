# Projeto-integrador - Levantamento de Requisitos
 <p>Este repositório contém os documentos porduzidos na disciplina de <b>Levantamento de Requisitos</b>, referentes ao desenvolvimento do sistema de gestão de estacionamento, <b>VagaCerta</b>. </p>
 <p>O obetivo é registrar, organizar e documentar as necessidades do cliente e, definir as funcionalidades do sistema, além de registrar cada passo de cada um no grupo do que foi feito, do que irá e o que está fazendo, documentado tudo no arquivo em docs, "daily". </p>

## 📝 Planilha de Requisitos
 Documento contendo todos os requisitos do software de gestão de estacionamento, incluindo:

- Requisitos Funcionais (RF);
- Requisitos Não Funcionais (RNF);
- Regras de Negócios (NF)
- Descrição detalhada do sistema;
  
 Planilha: https://docs.google.com/spreadsheets/d/1BVApUMaWUQPNNASIZNDYJ5TvKVKc6ieHBqPiTH9WILM/edit?usp=sharing
